# 📚 Sistem Perpustakaan Kampus (Terintegrasi API)

Aplikasi web sirkulasi peminjaman buku perpustakaan yang terintegrasi langsung dengan API **Pusat Data Mahasiswa** via Laravel Sanctum Token.
